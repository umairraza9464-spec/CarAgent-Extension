## Features

- **Auto Extract**: OLX/Facebook ke chat se automatically nikalo:
  - Customer Name
  - Mobile Number
  - Car Model
  - Registration Number

- **Manual Entry**: Directly add customer details
- **Data Storage**: Local storage mein sab data safe
- **Export CSV**: All data ko CSV mein download karo
- **Settings**: Google Sheet ID save karo

## How to Use

1. **Extract Mode**:
   - OLX ya Facebook chat kholo
   - Extension ke popup mein "Extract Now" click karo
   - Data automatically fill hoga
   - "Save" button click karo

2. **Manual Mode**:
   - Name, Mobile, Car Model enter karo
   - Save button click karo
   - Data local storage mein save hoga

3. **Export**:
   - Settings tab mein jao
   - "Export CSV" button click karo
   - CSV file download hoga
